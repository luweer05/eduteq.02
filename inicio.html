Escolha um módulo e comece a aprender

Cada módulo traz aulas em texto, um vídeo explicativo e desafios práticos. Estude na ordem que preferir — ou desafie-se direto no jogo.

HTML

HTML (HyperText Markup Language) é a linguagem que dá estrutura a todas as páginas da web. Aqui você aprende tags, semântica, formulários, mídia e boas práticas.
CSS

CSS (Cascading Style Sheets) é o que dá vida visual ao HTML. Cores, tipografia, espaçamento, layouts responsivos, animações — tudo passa por aqui.
JavaScript

JavaScript transforma páginas estáticas em experiências interativas. Aqui você aprende variáveis, funções, DOM, eventos e lógica de programação.
