CSS - O Design de Todo Site

O CSS (Cascading Style Sheets) é uma linguagem de estilo que define a aparência de páginas web, como cores, fontes e layout. Ele separa o conteúdo da estrutura visual, facilitando a criação de sites elegantes e consistentes.
O CSS permite criar páginas web visualmente atraentes, definindo elementos como cores, tipos de fonte, espaçamento entre elementos e até animações.

Estrutura Básica

Explicação:

Seletor → Elemento HTML a ser estilizado. Bloco de Declarações → Instruções de estilo aplicadas a um elemento HTML. Propriedades → São os atributos que definem como um elemento HTML será exibido no navegador. Valores → Uma informação atribuída a uma propriedade.

Formas de aplicar CSS

Externo → Arquivo .css vinculado via , ideal para reutilização. Interno → <p> style </p> no <p> head </p>, útil para páginas únicas. Inline → No atributo style do elemento, indicado apenas para ajustes pontuais.

Tipos de seletores comuns

Elemento → p (todos os <p>p</p>). Classe → .destaque (elementos com class="destaque"). ID →#principal (elemento com id="principal"). Atributo → img[src] (imagens com atributo src). Pseudo-classe →  a:hover (links quando o mouse está sobre eles).

Exemplos de CSS em Códigos

body { background-color: #f0f0f0; font-family: Arial, sans-serif; }

h1 { text-align: center; color: #00539f; }
