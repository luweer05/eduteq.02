HTML - A Base de Todo Site

HTML (HyperText Markup Language) é a linguagem utilizada para criar a estrutura de páginas web. Ele organiza e define os elementos que aparecem em um site, como títulos, textos, imagens, links, tabelas e formulários.
Pense no HTML como o esqueleto de uma página. Sem ele, não existiria conteúdo para ser exibido no navegador.

Estrutura Básica

Explicação:

<!DOCTYPE html> = Define a versão do HTML.
<html> = Início da página.
<head> = Parte de configurações do site.
<title> Meu Site </title> = Título da aba.
</head>
<body> = Tudo que aparece no site fica dentro do body.
</body>
</html>

Títulos

<h1>Título Principal</h1> Maior título da página.

<h2>Subtítulo</h2> h2 até h6.
<h3>Título menor</h3> Subtítulos.

Parágrafos

<p>Título Principal</p> Cria textos e parágrafos

Imagens

<img src="foto.png" alt="Imagem"> = srcCaminho da imagem alt = Texto alternativo

Links

<img src="imagem-exemplo.png" alt="Sua Descrição"> src = Caminho da imagem href = Define o destino do link
