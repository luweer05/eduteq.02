Laços de Repetição

for

for(let i=1; i<=5;i++){
console.log(i);
}
i=1 → começa em 1.
i <= 5 → repete até 5.
i++ → aumenta 1 a cada repetição.

while


                let i = 1; 
                
                while(i <= 5){
                    console.log(i)
                    i++;
                }

Repete enquanto a condição for verdadeira

do while


           let i = 1; 

           do{
            console.log(i);
            i++;
           } while(i <= 5);

Execute pelo menos uma vez.
